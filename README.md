Для того чтобы склеить отдельные файлы, которые мы получим из API Yandex Cloud
SpeechKit в один, нам понадобится `ffmpeg`.

Для MacOS его можно поставить
```shell
brew install ffmpeg
```

На Ubuntu 
```shell
apt install ffmpeg
```

C Windows как-нибудь сами разберетесь.
